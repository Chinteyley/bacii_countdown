# Bacii Exam Countdown

Live countdown for the Cambodian Grade 12 Bacii exam (target date: August 10, 2026 at 07:00 GMT+7). Available at https://bacii.ctey.dev.

## Features
- Real-time countdown with animated digits powered by NumberFlow.
- Animated hero copy using framer-motion.
- Responsive layout with Tailwind CSS v4 and Geist fonts.
- SEO metadata and Open Graph image configured in `src/app/layout.tsx`.

## Getting started
1) Install dependencies: `npm install`
2) Run the dev server: `npm run dev` and open http://localhost:3000
3) Lint the project: `npm run lint`
4) Build for production: `npm run build` then `npm run start`

To change the target exam date/time, update `targetDate` inside `src/app/page.tsx`.
